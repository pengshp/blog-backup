    CentOS 时间同步

    发表于 2018-01-31 更新于 2021-01-19 分类于 CentOS Valine：
    本文字数： 830 阅读时长 ≈ 1 分钟

    Linux中很多服务需要进行时间同步，不然容易导致出错。本文介绍使用阿里云的时间服务器同步服务器的时间。

    1、安装软件

    $ yum install -y ntpdate

    2、时间同步服务器

    可用的时间同步服务器，阿里提供了一些NTP时间服务器可以用于从互联网中同步服务器的时间；

    ntp1.aliyun.com
    ntp2.aliyun.com
    ntp3.aliyun.com
    ntp4.aliyun.com
    ntp5.aliyun.com
    ntp6.aliyun.com
    ntp7.aliyun.com

    3、同步时间

    从上面的时间同步服务器中选择一个进行时间同步。

    $ ntpdate -u ntp1.aliyun.com

    4、加入计划任务

    可以把时间同步加入系统的计划任务，定时从互联网同步时间

    $ crontab -e
* */2 * * * ntpdate -u ntp1.aliyun.com &> /dev/null
# 每两个小时同步一次时间
$ systemctl start crond
$ systemctl enable crond
